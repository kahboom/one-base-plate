# OneBasePlate — product context

**What it is:** A household meal-planning app built around **one base meal, multiple assemblies** — merged grocery lists and suggestions that respect picky eaters, texture needs, and prep constraints.

**What it is not:** A generic recipe manager. The product goal is one shared dinner that can be assembled differently per person, without running multiple parallel cooks.

**Bias when changing code:** Prefer practical, low-friction behavior on a tired weeknight (less decision fatigue, less waste, less defaulting to takeaway).
